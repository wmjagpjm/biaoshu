# P9B 国能 e 招计划追踪集成契约与验收记录

> **状态：** 已完成实现与独立验收（2026-07-13）
> **适用范围：** 仅适用于用户指定的“国能 e 招（国家能源招标网）”单站计划追踪；不构成对其他网站的接入授权。

## 1. 目标、范围与非目标

**目标：** 用户上传本机招标计划 .xlsx 后，服务端按固定规则查询国能 e 招招标公告，解析北京时间的投标截止/开标时间；用户人工确认后，才将命中写入既有本地标讯库。

**已实现范围：**

- 工作空间隔离的计划、同步运行和公告命中数据域；
- .xlsx 内存导入、原子校验与同空间幂等；
- 固定门户、固定检索请求、招标公告类别过滤、低频单条详情读取；
- 运行状态查询、仪表盘、动态公告链接和人工接受；
- 标讯页的国能计划追踪面板与本地 MockTransport 浏览器 E2E。

**明确非目标：**

- 不修改用户既有 08:30 Windows 计划任务、邮件、Excel 路径或桌面文件；
- 不提供应用内定时器、全站抓取、附件下载、登录、验证码、账号、代理或多站点配置；
- 不让浏览器直连国能 e 招，不接收 URL、Cookie、Token、主机或自定义检索条件；
- 不保存或回显 Cookie、原始 HTML/JSON、附件、请求头、远端异常原文或公告全文；
- 不自动创建技术标项目；未解析或待复核命中不可接受。

## 3. 固定网络与数据安全契约

### 3.1 网络边界与上限

- 唯一外部主机为 https://www.chnenergybidding.com.cn，请求由服务端固定客户端发起，禁用重定向。
- 固定门户为 https://www.chnenergybidding.com.cn/bidweb/；固定检索端点为 https://www.chnenergybidding.com.cn/bidfulltextsearch/rest/inteligentSearch/getFullTextData。它们是用户指定单站的受控读取路径，不宣称为正式开放 API。
- 同步开始时仅访问固定门户以获得匿名 uid；Cookie 仅保留在本次 httpx.Client 内存，任务结束即释放。
- 计划名是唯一检索输入；请求字段、门户路径和检索路径均由服务端固定，浏览器仅访问本机 /api。
- 单次最多 120 条计划、每计划最多 5 个候选、全运行最多 50 页详情；所有检索/详情请求最小间隔 1 秒。
- 连接超时 5 秒、读取超时 15 秒；单条检索最多重试 1 次，详情不重试。403、429 或连续两次网络错误停止后续读取。
- 只继续处理类别号以 001002 开头的招标公告；其他类别不读详情、不生成可接受命中。
- 详情地址绝不直接采信远端 linkurl：仅解析并校验 UUID、001002 类别和八位发布日期后，用固定 HTTPS 路径重建。

### 3.2 可存与禁止存储

| 对象 | 允许持久化 | 禁止持久化 |
|---|---|---|
| 追踪计划 | 计划名、招标人、范围、工期、预计发布时间文本、备注、指纹、启用状态、工作空间和时间戳 | 原始 Excel 字节、用户路径、帐号信息 |
| 同步运行 | 固定来源名、状态、开始/结束时间、脱敏数量和固定错误码 | URL、Cookie、请求/响应正文、堆栈和远端错误原文 |
| 公告命中 | 公告 ID、类别、发布日期文本、标题、关联计划、标准化时间、状态、已接受标讯 ID | HTML、全文、附件、Cookie、完整 JSON |
| 本地标讯 | 既有字段和不透明 chnenergy:公告ID 来源键 | URL、Token、Cookie、同步状态 |

announcementUrl 不入库，只在读取仪表盘时由 sourceInfoId、categoryNum 和发布日期文本动态生成。

## 4. HTTP、解析与人工确认契约

| 方法 | 路径 | 约束 |
|---|---|---|
| POST | /api/opportunity-watch/plans/import | 只接收 .xlsx；内存校验、失败零写入，返回导入计数。 |
| GET | /api/opportunity-watch/dashboard | 仅当前工作空间；返回计划数、最近运行和更新时间倒序命中。 |
| POST | /api/opportunity-watch/sync | 无请求体；只创建后台运行，返回 202 与 runId。 |
| GET | /api/opportunity-watch/runs/{runId} | 仅查询当前工作空间的运行；跨空间为 404。 |
| POST | /api/opportunity-watch/hits/{hitId}/accept | 仅 resolved 且有完整截止时间的命中可接受；重复调用不重复创建。 |

- 页面仅显示“国能 e 招候选公告，需人工确认；不会自动创建项目”。
- 解析仅从去标签后的可见文本提取完整 YYYY-MM-DD HH:mm:ss，固定 Asia/Shanghai；缺失或冲突时间为 needs_review。
- 接受时标题来自公告标题；采购人/摘要仅以本地计划字段补充，地区固定为“其他”；不从正文臆造预算、范围或采购人。
- 同步运行期间禁用同步；前端仅轮询本空间运行。只有 resolved 显示“加入本地标讯”，needs_review 不显示接受按钮。
- 前端外链只使用后端返回的 announcementUrl，并固定 target="_blank" 与 rel="noreferrer"。
- errorCode 仅可为 source_unavailable、rate_limited、malformed_response、interrupted 或空值；不得透传远端错误。
